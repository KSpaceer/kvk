Создание игры с помощью модуля pygame.
Жанр: beat 'em up
Суть: сюжета нет(или очень блеклый). Будет несколько уровней (допустим, пять), где главному персонажу предстоит сражаться с врагами нескольких видов (на первых уровнях будут более легкие, затем все больше и больше сложных).

План написания кода:


1. Создать экран																		+
2. Вывести на экран персонажа, который может стоять на месте и передвигаться в различных направлениях (до краев экрана) с "анимацией":				+
	2.1. При отсутствии перемещения будут активироваться два сменяющих друг друга изображения.								+					
	2.2. При перемещении будут активироваться два сменяющих друг друг изображения (шаг правой и шаг левой) в зависимости от направления			+
3. Вывести на экран врагов, которые будут появляться в случайной точке в дальнем от персонажа конце экрана(в правом или в левом), 
   после чего начнут передвижение в сторону игрока(сначала по вертикали, потом по горизонтали).									+
4. Добавить персонажу возможность атаковать при нажатии на пробел; при этом будет появляться прямоугольная поверхность, 
   соответствующая кулаку(или другому месту, которым совершается атака). 											+
   Атаку можно производить только раз в какое-то время. Например: анимация атаки длится секунду, поэтому в данное время нельзя совершить вторую атаку. 
   В коде можно реализовать так: пока существует "поверхность кулака", новая появиться не может. Соответсвенно с циклом анимаций.
	4.1 Сначала проигрывается анимация удара, во время которой появляется "поверхность кулака".								+
	4.2 После достижения максимальной дальности кулака и последнего кадра анимации, начинается анимация возвращения в исходное положение.
	    Поверхность удара при этом отсутствует. После возвращения в исходное положение можно совершить новый удар						+
5. Добавить врагам здоровье.																	+
6. При коллизии "поверхности кулака" с "поверхностью врага" количество здоровья этого врага будет снижаться. Если количество здоровья <= 0, 
   то начнет проигрываться анимация смерти. 															+
   В ином случае проигрывается анимация ударов.Т.к. я забыл снять как врагу наносится урон, можно сделать шатание. 						+
   Так же, если враг "умер", то коллизии кулака с поверхностью умершего врага учитываться не будут. 								+
   Во время получения урона враг не может совершать никаких действий.												+
7. Добавить врагам возможность атаковать при условии того, 													+
   что была достигнута необходимая дальность для соприкосновения "поверхности персонажа" с "ударной поверхностью". 						+
   Атака также может происходить с некоторой задержкой.														+
8. Добавить персонажу здоровье, выводящееся на экран. При получении удара от врагов это здоровье снижается, что отобразиться на экране. 			+			
   При получении урона будет проигрываться анимация, во время которой нельзя будет сделать никаких действий. 							+
   При снижении здоровья до нуля проигрывается анимация смерти													+
9. Начать делать уровни: добавить "волны" врагов, после победы над всеми волнами происходит переход на следующий уровень
10. Добавить главное меню, активирующееся при запуске:														+
	10.1. Кнопка "играть": запускает игровой процесс													+
	10.2. Кнопка "настройки": позволяет изменить разрешение экрана, сложность и пр. (когда будешь это добавлять, см. пункт "опционально")
	10.3. Кнопка "выйти": выключает игру															+

Опционально:

- Написать музыку и добавить звуки к действиям
- Перед главным меню добавить заставку, переход с которой к главному меню осуществляется нажатием любой клавиши							+
- Добавить возможность изменять разрешение. При изменении разрешения будут меняться размеры объектов
- Изменение сложности: увеличение здоровья и урона врагов, возможно - модификация волн врагов
- Нарисовать локации
